The Rolling Dice Cafe – Site Plan

Site Name

The Rolling Dice Cafe

This name captures the friendly, welcoming vibe of a local board game cafe while clearly signaling the board game theme. It is simple, fun, and memorable. A possible domain could be: rollingdicecafe.com.

Site Purpose

The Rolling Dice Cafe website serves as the digital hub for a local board game cafe. It provides essential information about the cafe, showcases its board game collection and menu, and promotes events such as game nights, tournaments, and community meetups. The site also encourages community engagement through RSVP links and contact forms.

Scenarios

  • What board games are available at the cafe, and can I filter them by type or number of players?
  • When is the next open game night or tournament, and how can I RSVP?

Color Schema

  • Primary Color: Navy Blue (#2e4057) – used for headings, body text, and structural accents.
  • Accent Color: Warm Orange (#f4a261) – used for buttons, highlights, and headings.

Typography

  • Headings: Playfair Display – a serif font for elegance and warmth.
  • Body: Quicksand – a clean, modern sans-serif font for readability and friendliness.

This document uses both fonts as they will be applied in the final project.
